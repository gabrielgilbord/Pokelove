# Pokélove — Ruta de Recuerdos (SPA)

App web de una sola página (HTML/CSS/JS) estilo **Pokémon RPG clásico** para móvil:

- **Overworld** en `canvas` (ruta con hierba alta, árboles, camino)
- Personaje caminando con **D-pad táctil** y **tap-to-move**
- 5 objetos brillantes (1→5) que se recogen **en orden**
- Escena de evento/batalla con **texto letra a letra**
- Final con **medalla + confeti pixelado + música de victoria**

## Cómo usar

- Abre `index.html` en el navegador.
- Activa **Sonido** si quieres efectos.
- Activa **Música** y coloca tus pistas en `assets/`.

## Personalizar fotos y textos

Edita `app.js` y cambia, para cada año:

- `photoUrl`: reemplaza `"Pon aquí la URL de tu foto"` por una URL pública HTTPS.
- `text`: pon tu recuerdo.

## Música de fondo

Por derechos de autor, aquí no se incluye música oficial de Pokémon.

- Coloca tu pista tranquila (ruta/pueblo) en:
  - `assets/route.mp3` o `assets/route.ogg`
- Coloca tu pista de victoria/final en:
  - `assets/victory.mp3` o `assets/victory.ogg`
